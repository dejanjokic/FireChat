# FireChat

FireChat is a native Android app built using Firebase's Machine Learning Kit, which makes it possible to send 8 different emoticons based on the user's face expression.

- Kotlin
- MVP
- Dagger2
- RxJava
- Firebase Auth
- Firestore
- Firebase ML Kit
